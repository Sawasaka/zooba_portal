# zooba Q&A スキル

ユーザーからの zooba に関する質問に対して、パートナーポータル（zooba_portal.html）のデータを参照し、正確に引用付きで回答します。

## 実行手順

1. `.claude/zooba_index.json` を Read ツールで読み込む
2. ユーザーの質問内容に関連するデータを検索する
3. 該当する情報をもとに回答を作成する
4. 必ず参照元を明記する

## データソース

インデックスファイル: `.claude/zooba_index.json`

### 含まれるデータセクション

| セクション | キー | 件数 | 内容 |
|-----------|------|------|------|
| FAQ | `faq` | 159件 | Q1〜Q159。質問・回答・カテゴリ・グループ |
| ヘルプセンター | `helpcenter` | 226件 | HC#1〜HC#226。記事タイトル・要約・カテゴリ・サブカテゴリ |
| 顧客課題一覧 | `issues` | 50件 | 1-01〜5-10。課題・ヒアリング項目・Zooba解決方法・営業Tips |

## 回答フォーマット

回答は以下のフォーマットで返す:

```
【回答】
（質問に対する回答内容）

📖 参照元:
- [FAQ Q{番号}] {カテゴリ} > {質問タイトル}
- [ヘルプセンター HC#{番号}] {カテゴリ} > {記事タイトル}
- [顧客課題 {番号}] {セクション名} > {課題タイトル}
```

## ルール

- 必ずインデックスデータに基づいて回答する（推測しない）
- 複数の関連情報がある場合はすべて引用する
- 該当する情報が見つからない場合は「ポータル内に該当する情報が見つかりませんでした」と明記する
- 回答は日本語で行う
- 引用元は具体的な番号・タイトルを記載する
